'''Bibledit''' is series of related [[open source software|open source]] software projects for editing and translating the [[Bible]]. Bibledit is free software.

== History ==

Bibledit [[Software development|development]] started in 2003.<ref name=start>[https://sites.google.com/site/bibledit/community/development Bibledit Development]</ref>

The word "Bibledit" is a portmanteau of the words "Bible" and "edit".

Bibledit was first hosted on [[SourceForge]].<ref name=sourceforge1>[https://sourceforge.net/projects/bibledit First Bibledit project on SourceForge]</ref>

== First generation ==

Bibledit-Gtk was the first generation of the software. It runs on the [[Desktop environment|desktop]].<ref name=bibleditgtk>[https://sites.google.com/site/bibledit Bibledit-Gtk documentation]</ref> Like all Bibledit programs, its source code is public,<ref name=bibleditgtksource>[https://github.com/teusbenschop/bibledit Bibledit-Gtk hosting on Savannah]</ref> and it can be freely downloaded.<ref name=bibleditgtkdownload>[https://download.savannah.gnu.org/releases/bibledit/source/gtk Bibledit-Gtk downloads]</ref>

== Second generation ==

The second generation of Bibledit focused on the [[World Wide Web|web]].

Bibledit-Web is a Bible editor that runs on the [[World Wide Web|web]], allowing users to edit the Bible from a [[web browser]]. The source code is public,<ref name=bibleditweb>[https://savannah.nongnu.org/projects/bibledit-web Bibledit-Web hosting on Savannah]</ref> and freely downloadable.<ref name=bibleditwebdownload>[https://download.savannah.gnu.org/releases/bibledit-web Bibledit-Web downloads]</ref>

Several version of the software have been built to work with other Bible software. Bibledit-Xiphos<ref name=bibleditxiphosdownload>[https://download.savannah.gnu.org/releases/bibledit/source/xiphos Bibledit-Xiphos downloads]</ref> and Bibledit-BibleTime<ref name=bibleditbibletimedownload>[https://download.savannah.gnu.org/releases/bibledit/source/bibletime Bibledit-BibleTime downloads]</ref> work with different versions of [[The SWORD Project]]. Versions also exist for BibleWorks,<ref name=bibleditbibleworksdownload>[https://download.savannah.gnu.org/releases/bibledit/binaries Downloads for Bibledit-BibleWorks]</ref> Paratext,<ref name=bibleditparatextdownload>[https://download.savannah.gnu.org/releases/bibledit/binaries Downloads for Bibledit-Paratext]</ref> and OnlineBible.<ref name=bibleditonlinebibledownload>[https://download.savannah.gnu.org/releases/bibledit/binaries Downloads for Bibledit-OnlineBible]</ref>

== Third and current generation ==

The third generation of Bibledit focuses on [[cloud computing]]. It is actively being developed and maintained. The source code is public.

Similar to Bibledit-Web, Bibledit Cloud on the web, allowing users to edit the Bible from a web browser.<ref name=bibleditlib>[https://github.com/teusbenschop/bibledit/tree/master/lib Bibledit Cloud hosting on github]</ref><ref name=bibleditclouddownload>[https://bibledit.org/cloud Bibledit Cloud downloads]</ref>

Offline versions are available for Windows,<ref name=windowsdownload>[https://bibledit.org/windows Downloads for Bibledit for Windows]</ref> Mac [[OS X]], [[Linux]], [[Android (operating system)|Android]], and [[IOS (Apple)|iOS]]. Users can edit the Bible from a web browser or app while off-line, and then synchronize the changes with Bibledit Cloud. The Android version is installed via [[Google Play]].<ref name=googleplay>[https://play.google.com/store/apps/details?id=org.bibledit.android Bibledit for Android on Google Play]</ref> iOS installation is by the [[App Store (iOS)|App Store]].<ref name=iosstore>[https://itunes.com/apps/bibledit Bibledit for iOS on the App Store]</ref>

== Usage ==

The Bibledit translation software is known to and used by people involved in Bible translation and production.

[[SIL International]] includes Bibledit in their list of software and fonts.<ref>https://www.sil.org/resources/software_fonts/bibledit</ref>

The [[Adapt It]] translation software can exchange files with Bibledit.<ref>https://www.adapt-it.org/Help_for_Administrators.htm</ref>

The Bible society of [[Baptist Mid-Missions]] has made a Bibledit Windows port for their translators.<ref>https://biblesint.org/biNew/bibledit</ref>

[[Pioneer Bible Translators]] wrote an in-depth review of Bibledit.<ref>https://map.bloomfire.com/posts/1041843-bibledit-now-translation-teams-can-edit-scripture-on-any-device</ref>
